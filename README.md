# 🟦🖊 Real-Time Whiteboard

A collaborative, real-time whiteboard built with *Vanilla JavaScript, HTML, and CSS, powered by **Socket.io* for seamless multi-user interaction.

## 🚀 Features

- 🎨 *Real-time Drawing*: Instantly reflect changes across multiple users using WebSockets.
- ✏ *Pencil & Eraser*: Adjustable size and color for precise sketching.
- 📝 *Sticky Notes & Images*: Add notes and images for better collaboration.
- 🔄 *Undo-Redo Functionality*: Stack-based approach to manage drawing history.
- 💾 *Download Board*: Save the whiteboard as an image.

## 🛠 Technologies Used

- *Frontend:* HTML, CSS, Vanilla JavaScript, Canvas API
- *Backend:* Node.js, Express.js
- *Real-time Communication:* Socket.io
